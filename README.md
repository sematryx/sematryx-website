# AEAO Website

The official website for AEAO (Agentic Expository Audidactic Optimizer) - the world's most advanced AI-powered optimization framework.

## Features

- **Landing Page**: Modern hero section with feature showcase
- **API Key Management**: Free API key generation + pay-per-execution billing via Stripe
- **Documentation**: Comprehensive docs with code examples and tutorials
- **Benchmarks**: Performance metrics and competitive analysis
- **MCP Integration**: Model Context Protocol documentation and examples

## Technology Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Payments**: Stripe integration
- **Deployment**: Vercel

## Getting Started

```bash
npm install
cp .env.example .env.local
# Configure environment variables
npm run dev
```

Visit `http://localhost:3000` to view the application.
