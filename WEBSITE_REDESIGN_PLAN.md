### Required Changes

#### Phase 1: Core Identity (Critical)
1. **Update Product Name Everywhere**
   - Metadata titles
   - Hero section
   - All page headers
   - Footer
   - README

2. **Update Hero Section**
   - Change from "automation platform" to "optimization framework"
   - Highlight the AEAO Tetrad
   - Show optimization use cases, not workflow automation

3. **Integrate Tetrad into Layout**
   - Make the 4 pillars visually prominent
   - Use tetrad structure in navigation
   - Feature each pillar with dedicated sections

#### Phase 2: Content Rewrite (High Priority)
1. **Features Component**
   - Replace with 4 Tetrad pillars
   - Add optimization-specific features
   - Remove automation language

2. **Documentation Pages**
   - Remove all automation API docs
   - Add optimization API documentation
   - Document tetrad configuration
   - Show optimization examples

3. **Tutorials**
   - Remove automation tutorials
   - Add optimization tutorials:
     - Basic optimization
     - Tetrad configuration
     - Domain-specific optimization
     - Advanced features

#### Phase 3: API Documentation (High Priority)
1. **Remove**
   - `/docs/api/automation` - wrong product
   - `/docs/api/webhooks` - wrong product
   - All webhook/trigger/action references

2. **Add**
   - `/docs/api/optimization` - main optimization API
   - `/docs/api/tetrad-config` - tetrad configuration
   - `/docs/api/domains` - domain-specific APIs
   - `/docs/api/explainability` - expository intelligence
